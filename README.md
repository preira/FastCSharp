# FastCSharp
Build Fast Code using a set of patterns that simplify building REST and MQ services.  
FastCSharp contains several libs that speed up the development.  
Currently, you can find the following accelerators:  
* [Circuit Breaker](./CircuitBreaker/README.md) - A flexible circuit breaker implementation that let's you choose from and combine different backoff, breaker and circuit handling strategies.    
* [Publisher](./Publisher/README.md) - provides a SDK to implement publishers for specific engines such as RabbitMQ.  
* [Subscriber](./Subscriber/README.md) - provides a SDK to implement subscriber for specific engines such as RabbitMQ.
